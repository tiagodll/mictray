Why MicTray?

🔴

Visual Peace of Mind

A bright red icon lets you know exactly when your mic is hot. A subtle gray means you are safely muted.

⚡️

One-Click Toggle

Never scramble for the Zoom or Teams mute button again. Just click the tray icon to mute or unmute globally in an instant.

🛠

Deep OS Integration

MicTray directly speaks to macOS via AppleScript to adjust the system-level input volume. It works universally across all your apps.

How to Install and Use

1

Download or Build

Download the latest MicTray.zip from GitHub Releases, extract it, and drag MicTray.app to your Applications folder. Alternatively, you can build it yourself from source:

git clone https://github.com/tiagodll/mictray
cd mictray
make
2

Run MicTray

Launch MicTray.app from your Mac's Applications folder or Spotlight to see it appear in your macOS menu bar instantly. (It runs entirely in the background without resting in your Dock!)

3

Click and Control

Left-click the tray icon to instantly toggle your volume. Right-click the icon to open a handy context menu to view the About page or Quit the application cleanly.
